Question Link : https://leetcode.com/problems/evaluate-reverse-polish-notation/description/

*   Problem:
        Evaluate Reverse Polish Notation (Postfix expression)

*   Example:
        ["2", "1", "+", "3", "*"]

        → ((2 + 1) * 3) = 9

*   Key idea:

        Use STACK

*   Why stack?

        Because postfix evaluation is naturally LIFO

*   Algorithm:

        Traverse tokens:

        If number:
            push to stack

        If operator:
            pop two elements
            apply operation
            push result back

*   Operand order:

        VERY IMPORTANT:

            num1 = stack.pop() → right operand
            num2 = stack.pop() → left operand

        For:
            - → num2 - num1
            / → num2 / num1

*   Why?

        Stack pops in reverse order

*   Stack state example:

        tokens = ["2","1","+"]

        stack:
            push 2
            push 1

        encounter "+"
            pop 1 (num1)
            pop 2 (num2)

            result = 2 + 1

*   Time Complexity:

        O(n)

*   Space Complexity:

        O(n)
